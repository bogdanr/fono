# Fono

Talk to Linux — press a key, speak, and Fono types it into any window, routes it
to your LLM, or drives your coding agent by voice. Local-first, one static binary.

Install the prebuilt binary with [`cargo-binstall`](https://github.com/cargo-bins/cargo-binstall):

```sh
cargo binstall fono
```

Or use the install one-liner:

```sh
curl -fsSL https://fono.page/install | sh
```

> This crates.io package is a lightweight stub that provides `cargo binstall`
> metadata. The application itself is distributed as prebuilt binaries on
> GitHub. Homepage: <https://fono.page> · Source: <https://github.com/bogdanr/fono>
