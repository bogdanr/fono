## What landed in the last session (Phase 9 + 10)

* **release.yml**: staged per-target artifact dirs; real macOS lipo into a
  universal binary; optional minisign signing of SHA256SUMS controlled by
  the `MINISIGN_KEY` repo secret.
* **NimbleX SlackBuild** at `packaging/slackbuild/fono/` mirroring the
  earlyoom template. Two modes — binary (default, fetches the release
  tarball) and source (`FROM_SOURCE=1`). Installs `/usr/bin/fono`, desktop
  entry + SVG icon, per-user `fono.service`, and bash/zsh/fish completions
  generated by the new `fono completions <shell>` subcommand.
* **Systemd unit hardened**: `NoNewPrivileges`, `ProtectSystem=strict`,
  `ProtectHome=read-only`, `ReadWritePaths=%h/.config/fono %h/.local/…`.
* **Parallel packaging recipes**: `packaging/aur/PKGBUILD`,
  `packaging/nix/flake.nix`, `packaging/debian/{control,rules,...}`.
* **Docs filled in**: README got install matrix + CLI + quickstart + disk
  layout; `docs/providers.md` (STT/LLM matrix + env-var names),
  `docs/architecture.md` (module map + runtime dataflow + FSM + paths),
  `docs/privacy.md` (what stays vs. leaves, retention, deletion),
  `docs/wayland.md` (sway/hyprland vs GNOME/KDE portal caveats).

Build status: `cargo build --workspace` ✅, `cargo test --workspace --lib`
→ **26 passed / 0 failed**, `cargo clippy --workspace --no-deps -- -D
